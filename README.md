This repo stores Group 10's ELEC 376 project. 

What is the project idea?
Create a user-friendly web/mobile platform where renters can share and read reviews of landlords. The project will help tenants make informed housing decisions, improve rental transparency, and encourage better landlord practices.

What is the project scope?
User account creation and login (profile information). 
Search and filter landlords by name, address, or city. 
As a secondary, record every property this landlord is associated with 
Post and edit reviews with ratings and optional photos. 
View aggregated ratings and comments. 
Landlord leaderboards (within the local area). 
Basic reporting of inappropriate content (censoring keywords, and offensive phrases). 
Basic validation of comments on landlords, or disclaimers on the validity of information on the landlord. 

What is the intended audience?
Tenants and renters will be the intended users
