# Rate My Landlord  
A full stack rental review platform built by Group 10 for ELEC 376. This platform helps students and tenants share experiences, rate landlords, and browse reviews in a simple and transparent way.

## Overview  
Rate My Landlord allows users to:
• Create accounts and log in  
• Browse landlords and property listings  
• Submit reviews with text and ratings  
• View a landlord leaderboard sorted by score  
• See detailed landlord profiles and past reviews  

The goal of the project is to improve rental transparency and help tenants make informed housing decisions.

## Setup Instructions  

1. Clone the repository  
git clone https://github.com/armaansingla14/elec376-landlord-app.git  
cd elec376-landlord-app  

2. Start the backend  
cd backend  
npm install  
npm start  

3. Start the frontend  
cd ../frontend  
npm install  
npm start  

Frontend runs on port 3000.  
Backend runs on port 5000.
